# OBS Floating Ball

适用于 Windows 的 OBS 本机录制悬浮控制器。它通过 OBS WebSocket 连接本机 OBS，将开始、暂停、继续、停止录制与最近文件重命名放到可拖动的小型控制条中。

当前版本：[v0.1.0](../../releases/tag/v0.1.0)

## 功能

| 功能 | 说明 |
| --- | --- |
| OBS WebSocket 连接 | 连接本机 `127.0.0.1:4455` 的 OBS WebSocket 服务，并在断线后提示连接状态。 |
| 录制控制 | 支持开始录制、暂停、继续和停止；按钮可用性会随 OBS 的实际状态更新。 |
| 录制计时与状态色 | 悬浮球显示录制时长，并以蓝、红、橙、绿、灰区分准备、录制、暂停、完成和未连接状态。 |
| 最近文件重命名 | OBS 返回录制文件路径后，可通过铅笔按钮为最近一次录制重命名，并校验非法文件名和重名。 |
| 悬浮控制条 | 点击计时球展开或收起控制条；拖动计时球或控制条可移动位置；右键可隐藏。 |
| 托盘与快捷键 | 托盘菜单可显示/隐藏、打开连接设置和退出；`Ctrl+Alt+H` 可切换悬浮控制条。 |
| 中英文界面 | 连接设置中可切换中文或 English。 |
| 本地安全存储 | OBS WebSocket 密码使用当前 Windows 用户的 DPAPI 加密后保存在 Qt 用户设置中。 |

## 系统要求

- Windows 10 19045 或更高版本。
- OBS Studio 28 或更高版本。
- OBS 的 WebSocket 服务已启用。OBS 28+ 已内置 obs-websocket 5.x。
- 仅支持本机 OBS：服务地址固定为 `127.0.0.1:4455`。

## 安装与首次连接

1. 前往 [Releases](../../releases/latest) 下载 `OBS.Floating.Ball.exe`。
2. 启动 OBS，打开“工具 -> WebSocket 服务器设置”。
3. 勾选“启用 WebSocket 服务器”，保持端口为 `4455`，并设置认证密码。
4. 运行 `OBS.Floating.Ball.exe`。首次运行会打开“OBS 连接设置”。
5. 输入与 OBS 一致的 WebSocket 密码，选择界面语言后点击“保存并连接”。
6. 连接成功后，屏幕右上角会出现悬浮球。连接失败时，请确认 OBS 正在运行、WebSocket 服务已启用且密码正确。

## 使用说明

### 悬浮球与控制条

1. 点击计时球展开或收起控制条。
2. 拖动计时球或控制条的空白区域可改变悬浮位置。
3. 右键计时球或控制条，选择“隐藏悬浮球”可暂时隐藏；按 `Ctrl+Alt+H` 或点击托盘图标可再次显示。
4. 展开后的三个操作依次为：开始/暂停/继续、停止录制、重命名最近录制文件。

播放三角在录制期间仍保持与设计样例一致的外观；其悬浮提示会明确显示当前操作是“开始录制”“暂停录制”或“继续录制”。停止按钮仅在 OBS 正在录制或暂停时可用。铅笔按钮会在录制结束且 OBS 已返回输出文件路径后启用。

### 状态颜色

| 颜色 | 状态 | 含义 |
| --- | --- | --- |
| 蓝色 | 已连接、待录制 | 已连接到 OBS，可以开始新的录制。 |
| 红色 | 正在录制 | 计时器持续更新；可暂停或停止。 |
| 橙色 | 已暂停 | 计时器保持不变；可继续或停止。 |
| 绿色 | 已结束 | 最近一次录制已完成，可重命名输出文件。 |
| 灰色 | 未连接或连接异常 | 请检查 OBS、WebSocket 服务和密码。 |

### 重命名录制文件

1. 在 OBS 中停止录制，等待悬浮球变为绿色。
2. 点击铅笔按钮。
3. 输入新文件名，不要包含 `\\ / : * ? " < > |`。
4. 程序会保留原文件扩展名，并阻止覆盖已有文件。

## 录屏与采集说明

悬浮控制条为了获得平滑圆角，使用逐像素透明窗口。因此，它会出现在 OBS 的“显示器采集”画面中；这是当前视觉优先设计的已知限制。

项目内的桌面标注基础组件仅能由“显示器采集”录入，Windows 的“窗口采集”和“游戏采集”不会录入桌面标注层。标注工具面板可使用 `WDA_EXCLUDEFROMCAPTURE` 尝试从采集画面中排除，但是否可用取决于 Windows 与采集方式。

## 常见问题

### 显示“无法连接 OBS”或“密码错误”

确认 OBS 正在运行；在“工具 -> WebSocket 服务器设置”中启用了服务；端口仍为 `4455`；输入的认证密码与 OBS 设置一致。修改密码后，重新打开托盘菜单中的“连接设置”保存即可。

### 悬浮控制条不见了

按 `Ctrl+Alt+H`，或单击系统托盘中的 OBS Floating Ball 图标以显示控制条。若正在使用全屏程序，先切回桌面确认其位置。

### 为什么控制条出现在录制画面中

这是平滑透明边缘与 Windows 采集排除机制之间的取舍。当前版本优先保证圆角和玻璃质感，因此悬浮条会出现在“显示器采集”中；录制前请将控制条移动到不会遮挡内容的位置，或使用快捷键暂时隐藏。

### 铅笔按钮不可用

只有停止录制、OBS 已报告输出文件路径且文件仍存在时，重命名功能才会启用。

需要 Python 3.11 或更高版本：

```powershell
py -3.11 -m pip install -e ".[dev]"

py -3.11 main.py
```

```powershell
.\build.ps1
```
