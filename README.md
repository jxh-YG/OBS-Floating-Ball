# OBS Floating Ball

适用于 Windows 的 OBS 本机录制悬浮控制器。它通过 OBS WebSocket 连接本机 OBS，将开始、暂停、继续、停止录制与最近文件重命名放到可拖动的小型控制条中。

当前版本：v0.3.0

## 功能

| 功能 | 说明 |
| --- | --- |
| OBS WebSocket 连接 | 可配置本机地址与端口（默认 `127.0.0.1:4455`），断线自动重试，鉴权失败引导设置。 |
| 录制控制 | 开始 / 暂停 / 继续 / 停止；主按钮录制中显示暂停图标。 |
| 全局快捷键 | 默认：显示隐藏 `Ctrl+Alt+H`，开始 `Ctrl+Alt+R`，暂停继续 `Ctrl+Alt+P`，停止 `Ctrl+Alt+S`；均可在设置中修改。 |
| 录制计时与状态色 | 蓝/红/橙/绿/灰表示就绪、录制、暂停、完成、未连接；录制中红球脉冲。 |
| 最近文件 | 记住最近录制，支持重命名、打开文件夹、最近列表；可选停止后自动弹出重命名（含占用重试）。 |
| 悬浮控制条 | 点击展开；含开始/暂停、停止、重命名；拖动记忆位置；可录制时自动隐藏。 |
| 托盘与单实例 | 托盘菜单；状态色图标；防止重复启动。 |
| 中英文界面 | 设置中可切换中文或 English。 |
| 本地安全存储 | WebSocket 密码经当前 Windows 用户 DPAPI 加密保存。 |
| 开机启动 | 可选登录时启动。 |

## 系统要求

- Windows 10 19045 或更高版本。
- OBS Studio 28 或更高版本（内置 obs-websocket 5.x）。
- 本机 OBS WebSocket 已启用。

## 安装与首次连接

1. 前往 [Releases](../../releases/latest) 下载 `OBS.Floating.Ball.exe`。
2. 启动 OBS，打开“工具 -> WebSocket 服务器设置”。
3. 启用服务，设置端口与可选密码。
4. 运行程序，首次填写连接信息、快捷键与偏好后“保存并连接”。

## 使用说明

### 悬浮球

1. 点击计时球展开/收起控制条。
2. 拖动后位置会自动保存。
3. 右键：打开录制文件夹、隐藏悬浮球。
4. 展开按钮：开始/暂停/继续、停止、重命名。

### 状态颜色

| 颜色 | 状态 |
| --- | --- |
| 蓝色 | 已连接、待录制 |
| 红色 | 正在录制（脉冲） |
| 橙色 | 已暂停 |
| 绿色 | 已结束，可重命名 |
| 灰色 | 未连接 |

### 录屏与采集说明

悬浮控制条为实心胶囊，会尝试用系统能力从“显示器采集”中排除。若仍进画面，可开启“录制时隐藏悬浮球”或用快捷键临时隐藏。

## 常见问题

- **无法连接 / 密码错误**：确认 OBS 与 WebSocket、地址端口与密码；空密码亦可。
- **悬浮球不见了**：使用显示/隐藏快捷键或托盘图标。
- **控制条进画面**：实心胶囊通常可采集排除；若系统不支持可录制时自动隐藏。
- **快捷键无效**：可能被占用，到设置中修改或“恢复默认快捷键”。

```powershell
py -3.11 main.py
```
